# AI Agents

This folder contains shared AI agent definitions and prompt assets.

## Layout

Domain folders:

- `01-core-development`
- `02-language-specialists`
- `03-infrastructure`
- `04-quality-security`
- `05-data-ai`
- `06-developer-experience`
- `07-specialized-domains`
- `08-business-product`
- `09-meta-orchestration`
- `10-research-analysis`

## Usage

Point each tool's agent directory to this folder if that tool supports
user-level agent definitions.
